# Назва проєкту
Seshn Networks


## Опис проекту
Проект призначений для спрощення комунікації між учасниками заходів або коворкінгів. Основна функція — можливість надіслати пряме повідомлення користувачу, який знаходиться поруч, з метою запрошення на спільну каву або коротке обговорення бізнес-ідей. Програма автоматично формує текст запиту, враховуючи локацію та спільні інтереси сторін.
